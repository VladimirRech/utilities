FUNCIONALIDADES A SEREM IMPLANTADAS

Criar agenda genérica que permita criar eventos com os seguintes dados:

  1. Data e hora de início de um evento
  2. Data e hora de término do envento
  3. Proprietário do evento (somente o nome na primeira versão)
  4. Título
  5. Descrição
  
Operações

  1. Listar eventos de um usuário em um período de tempo
  2. Recuperar um evento específico pelo id
  3. Inserir um evento novo
  4. Alterar dados do evento pelo id
  5. Excluir evento pelo id
